# 暖伴勤工 · 客人演示指南

## 客人（公网 · 固定链接）

**直接打开下面链接**（无需安装、无需克隆、无需连 WiFi）：

```
https://jushuolot.github.io/jinshouzhi/nuanban/#/pages/common/login
```

1. 微信 / 手机浏览器 / 电脑浏览器打开  
2. 输入手机号 → **登录**（演示任意 11 位均可；验证码可留空）  
3. 首次登录 → **选择身份**（学生 / 家属 / 老人）  

> **公网演示模式**：数据在浏览器内模拟，无需后端。地图 H5 可能空白，**列表模式可完整演示**。  
> **电脑打开**：顶部可切换 **iPhone / 华为 X6** 预览框；手机打开为全屏。

## 按角色演示路径

登录页底部 **测试账号** 可点选填入；验证码可留空。

| 手机号 | 角色 | 可测场景 |
|--------|------|----------|
| 13800000001 | 学生（林同学 · 主流程） | 接单、签到、完成服务、收入、推荐有奖 |
| 13800000002 | 学生（周同学 · 城东师范） | 发现页「学校合作」筛选、合作机构老人 |
| 13800000003 | 学生（待审同学） | 审核中页、无法接单 |
| 13800000004 | 家属 | 绑定老人、代付、外出审批、SOS、服务包 |
| 13800000005 | 老人 | 找陪护、预约、家属绑定码、一键求助 |

> **机构派单**：登录页 **更多 → 超级管理** → 机构派单（无独立演示手机号）。

### 学生端 `13800000001`

| 步骤 | 操作 |
|------|------|
| 1 | 登录 → **首页** 看统计、待接单角标 |
| 2 | 底部 **接单** → 富订单卡片 → 进入详情 → **接受订单** |
| 3 | **服务中** / **排班** → **到场签到**（500m 围栏，演示定位自动通过）→ **完成服务** |
| 4 | **发现** → 附近老人 PersonCard → 老人详情 |
| 5 | 首页点 **本月收入** 或 **我的 → 收入明细** |
| 6 | 若老人触发 SOS，首页出现 **紧急求助** 待办 |

### 家属端 `family1@test.nuanban.dev`

| 步骤 | 操作 |
|------|------|
| 1 | 登录 → **首页** 待办统计；**绑定老人**（绑定码 / 列表） |
| 2 | **订单** → 详情看时间轴 → 待支付去 **模拟支付** |
| 3 | **外出审批** → 同意 / 拒绝 |
| 4 | 老人 **一键求助** 后，首页 **SOS** 待办 → 确认已知晓 |

### 老人端 `elder1@test.nuanban.dev`

| 步骤 | 操作 |
|------|------|
| 1 | 登录 → **首页** 统计与快捷入口 |
| 2 | **找陪护** → 学生 PersonCard → 详情 → 预约服务 |
| 3 | **我的 → 家属绑定码** 复制给家属扫码/输码 |
| 4 | **我的服务** → 订单详情（时间轴）；**一键求助** → SOS 待办 |
