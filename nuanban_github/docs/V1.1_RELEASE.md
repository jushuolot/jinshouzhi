## 相对 v1.0 变更摘要

### 体验与界面（v1.1）

- 三端首页（学生 / 老人 / 家属）统计卡统一 `nb-stats-card` / `nb-section-title` 主题类
- 登录页角标与表单色统一 `theme.css` 变量（暖色品牌一致）
- 运营概览页动态区文字色改为主题变量
- 家属首页：绑定老人空状态卡片化，文案修正
- 老人预约页：无 SKU 时展示运营指引空状态（机构 → 服务目录）

### 版本与文档

- 全站版本字符串 v1.0 → v1.1
- 新增 `V1.1_VERIFICATION_CHECKLIST.md` 逐项验收清单
- 本发布说明

### 延续 v1.0 能力（未回退）

- 正式鉴权、PocketBase 真实数据路径
- 运营：服务目录、学校合作、派单、老人档案
- 资金持久化：`wallet_accounts` / `wallet_transactions` / `payment_accounts` / `withdrawals`
- `isDemoMock` 门禁演示字符串

## 已知限制（v1.2 候选）

- 微信登录 / 支付仍为演示或 stub
- 结算历史订单 backfill 脚本可选，未默认执行
- 独立 Admin Web 仍由小程序运营台替代
