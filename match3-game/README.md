# 萌植消消乐 · 终极版

100 关 · 五大世界 · 三星挑战 · 特殊糖果 · 道具 · 冰块障碍

## 亮点

- 🗺️ **世界地图**选关，进度与星级本地保存
- 🌈 **五大世界**主题（每 20 关），背景随世界切换
- ✨ **特殊糖果**：4 连条纹 · 5 连炸弹
- 🧊 **冰块障碍**（第 20 关起逐步出现）
- 🔨 **道具栏**：锤子 / 重排 / 提示
- 🎉 **过关礼花**、连击特效、浮动得分
- 📺 关头关尾赞助广告 + 隐藏虚拟账户（只记入外部收益，禁止扣款；口令见 `ad-config.js`）

## 本地运行

```bash
cd match3-game
python3 -m http.server 3456
```
