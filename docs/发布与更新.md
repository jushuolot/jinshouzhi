# 发布与更新（仅 GitHub + 本机）

> **推荐云端演示：GitHub Codespaces**。不需要 Render，不需要公网部署。

仓库：https://github.com/jushuolot/jinshouzhi

---

## 一、本地改代码

项目目录：

```text
~/Downloads/cursor/jinshouzhi
```

用 Cursor / VS Code 编辑后，在本机终端自测：

```bash
cd ~/Downloads/cursor/jinshouzhi
npm run dev
```

浏览器：**http://localhost:5173**

---

## 二、同步到 GitHub（必做）

```bash
cd ~/Downloads/cursor/jinshouzhi
git add .
git commit -m "说明你改了什么"
git push
```

推送成功后，在网页可看到最新代码：  
https://github.com/jushuolot/jinshouzhi

---

## 三、在 GitHub Codespaces 里看到最新版

### 方式 A：已有 Codespace

1. 打开 https://github.com/codespaces  
2. 进入你的 **jinshouzhi** 环境  
3. 终端执行：

```bash
git pull
npm run dev
```

4. **端口 5173** → 在浏览器中打开

### 方式 B：重新创建（最干净）

1. https://github.com/jushuolot/jinshouzhi  
2. **Code → Codespaces → Create codespace on main**  
3. `npm run dev` → 打开 **5173**
