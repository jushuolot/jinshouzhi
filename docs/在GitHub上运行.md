# 在 GitHub 上运行「金手指」

## 推荐：方案 A（Codespaces）

**在浏览器里运行，不用本机杀端口。**

👉 **完整步骤请看：[方案A-Codespaces从零开始.md](./方案A-Codespaces从零开始.md)**

5 步极简：

1. https://github.com/jushuolot/jinshouzhi → **Code → Codespaces → Create**
2. 等环境建好
3. 终端：`npm run dev`
4. 打开端口 **5173**
5. 登录 `13800001001` / `123456`
