# 在 GitHub 上运行「金手指」

## 唯一推荐的云端方式：GitHub Codespaces

**在浏览器里运行，不用本机杀端口，不需要 Render。**

👉 **完整步骤：[方案A-Codespaces从零开始.md](./方案A-Codespaces从零开始.md)**

一键创建：https://github.com/codespaces/new?repo=jushuolot/jinshouzhi

5 步极简：

1. 打开仓库 → **Code → Codespaces → Create codespace on main**
2. 等环境建好（自动安装依赖）
3. 终端：`npm run dev`
4. **端口 5173** → 在浏览器中打开（勿直接开 3001）
5. 登录 `13800001001` / `123456`
