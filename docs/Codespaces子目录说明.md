# Codespaces 目录说明

仓库根目录即为各子项目所在位置，**无需**再 `cd` 进嵌套的 `jinshouzhi/` 目录。

## 子项目目录

| 目录 | 说明 |
|------|------|
| `nuanban_github/` | 暖伴勤工 |
| `stock-assistant/` | Streamlit 股票助手 |
| `python-stock-mini/` | 轻量行情页 |
| `lo-delivery-platform/` | 物流订单平台 |
| `match3-game/` | 三消游戏 |

## 暖伴示例

```bash
cd nuanban_github
./scripts/dev-test.sh
./scripts/start-h5.sh
```
