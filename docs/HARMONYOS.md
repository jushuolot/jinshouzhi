# 金手指 · 鸿蒙系统使用指南

本项目的 **鸿蒙支持以 H5 为主**：在鸿蒙手机的系统浏览器或 WebView 中打开网页即可使用，无需先上架应用市场。

---

## 一、推荐方式：鸿蒙浏览器访问（最快）

### 1. 在电脑上启动服务

```bash
cd jinshouzhi
npm run install:all
npm run seed
npm run dev
```

保持两个窗口运行（或根目录 `npm run dev` 同时启动前后端）。

### 2. 查电脑局域网 IP

- macOS：系统设置 → 网络 → 查看 IP（如 `192.168.1.100`）
- Windows：`ipconfig` 查看 IPv4

### 3. 鸿蒙手机连接同一 WiFi

在手机浏览器（华为浏览器 / 系统浏览器）地址栏输入：

```text
http://192.168.1.100:5173
```

把 IP 换成你的电脑 IP。

> 开发模式用 **5173**（前端）；若只启动了后端且已 `npm run build`，可访问 `http://IP:3001`。

### 4. 登录测试

## 二、添加到桌面（像 App 一样打开）

### 方式 A：PWA（若浏览器支持「添加到桌面」）

1. 用浏览器打开上述地址  
2. 菜单 → **添加到主屏幕** / **添加桌面快捷方式**  
3. 桌面图标名称：**金手指**

> 鸿蒙不同版本浏览器对 PWA 支持不一致；若无此选项，用方式 B。

### 方式 B：浏览器书签 / 快捷方式

将网址存为书签，或拖到桌面快捷方式（视浏览器而定）。

## 三、常见问题

| 问题 | 处理 |
|------|------|
| 手机打不开网页 | 确认手机和电脑 **同一 WiFi**；电脑防火墙允许 5173、3001 端口 |
| 能开页面但登录失败 | 开发时用 **5173** 入口（带 API 代理），不要只开 3001 又没 build |
| 输入框被键盘挡住 | 已做 `safe-area` 与 `100dvh` 适配；可尝试竖屏 |
| 页面字体偏大/偏小 | 已禁用系统随意放大字体；可在系统设置调整显示大小 |

## 五、限制说明
