# 金手指 — 从零开始（小白版）

按顺序做，**不要跳步**。每一步完成后再做下一步。

---

## 第 0 步：关掉乱七八糟的终端

1. 打开「终端」应用。
2. 若有窗口在跑 `npm run dev`，点一下那个窗口，按 **`Ctrl + C`** 停掉。
3. 可以只留 **一个** 终端窗口备用。

---

## 第 1 步：进入项目文件夹（最重要）

复制下面 **一整行**，粘贴到终端，按回车：

```bash
cd /Users/chenli/Downloads/cursor/jinshouzhi
```

检查是否成功：

```bash
pwd
```

必须显示：

```text
/Users/chenli/Downloads/cursor/jinshouzhi
```

再检查有没有这些文件：

```bash
ls
```

应看到：`package.json`、`server`、`client`、`README.md`。

> **常见错误**：在 `chenli@Leon-Mac ~ %`（家目录）直接 `npm install` 会报错 `ENOENT package.json`。一定要先 `cd` 进上面路径。

---

## 第 2 步：安装依赖（第一次要做，约 2～5 分钟）

**确保还在 jinshouzhi 目录**，复制粘贴：

```bash
npm install
cd server && npm install && cd ..
cd client && npm install && cd ..
```

没有红色 `ERROR` 即可继续。

---

## 第 3 步：写入测试账号

```bash
npm run seed
```

成功会看到：

- 男士 `13800001001`，邀请码 `INV_M001`
- 女士 `13900002001`
- 密码都是 **`123456`**

---

## 第 4 步：启动项目（保持这个窗口开着）

```bash
npm run dev
```

应看到类似：

```text
[server] 金手指 API: http://localhost:3001
[client] Local: http://localhost:5173/
```

**不要关这个窗口。** 要停止时按 `Ctrl + C`。

---

## 第 5 步：浏览器打开

1. 打开 Chrome / Safari。
2. 地址栏输入：**http://localhost:5173**
3. 登录：
   - 手机号：`13800001001`
   - 密码：`123456`

登录成功会进入首页，可点 **「系统分配」**。

---

## 第 6 步：检查后端是否正常（登录失败时做）

**新开一个终端**（不要关第 4 步那个），执行：

```bash
curl http://localhost:3001/api/health
```

应返回：`{"ok":true,"product":"金手指"}`

若 `Connection refused`：回到第 4 步，确认 `npm run dev` 还在跑且无红色报错。

---

## 如何停止

在跑着 `npm run dev` 的终端按 **`Ctrl + C`**。

## 故障排除

### 1. `Could not read package.json` / 路径是 `/Users/chenli/package.json`

**原因**：没进项目目录。  
**解决**：重新执行第 1 步 `cd .../jinshouzhi`。

### 2. `Cannot open database because the directory does not exist`

**原因**：数据库目录未创建（已修复，更新代码后重试）。  
**解决**：

```bash
cd /Users/chenli/Downloads/cursor/jinshouzhi
mkdir -p data
npm run seed
npm run dev
```

### 2b. `SQLITE_READONLY` / `readonly database`

**原因**：旧数据库文件损坏或被占用。  
**解决**：先 `Ctrl+C` 停掉 `npm run dev`，再执行：

```bash
cd /Users/chenli/Downloads/cursor/jinshouzhi
rm -f data/jinshouzhi.db data/jinshouzhi.db-*
npm run seed
npm run dev
```

### 3. 登录只显示「失败」或「无法连接服务器」

| 检查 | 做法 |
|------|------|
| 后端没启动 | 第 4 步 `npm run dev` 是否在跑 |
| 没测试账号 | 第 3 步 `npm run seed` |
| 密码错 | 用 `123456`，不要有空格 |

### 4. `cd: no such file or directory: server`

**原因**：不在 `jinshouzhi` 目录。回到第 1 步。

### 5. Node 版本

推荐 **Node 20 LTS**。你若是 Node 25，一般也可用。查看版本：

```bash
node -v
```

## 鸿蒙手机访问（可选）

1. 手机和电脑同一 WiFi。
2. 电脑查 IP：`ipconfig getifaddr en0`
3. 手机浏览器打开：`http://你的IP:5173`（例如 `http://192.168.1.100:5173`）

详见 `docs/HARMONYOS.md`。
