# Stock Assistant · 产品说明

> **Stock Assistant（股票助手）** 是一款在浏览器里运行的股票分析小工具。  
> 适合个人学习、内部分享、快速看行情——**不是**专业交易终端，也**不能**代替券商或投顾。

---

## 1. 产品定位

| 维度 | 说明 |
|------|------|
| **是什么** | 基于 [Streamlit](https://streamlit.io) 的网页应用：搜索股票 → 加入自选 → 看 K 线 / 板块 / 榜单 → 生成可读简报 |
| **给谁用** | 需要快速了解 A 股、港股、美股行情的同学；想把分析整理成 Markdown 发给同事的人 |
| **不是什么** | 不提供下单、不保证实时行情、不构成投资建议 |
| **当前阶段** | **P8**（推送 + 多用户）已完成，详见 [EVOLUTION.md](../EVOLUTION.md) |

**一句话**：三步完成分析——**发现标的 → 工作台分析 → 导出可读简报**。

---

## 2. 功能地图

侧边栏有「快速上手」指引；主界面按使用顺序分为 7 个标签页：

| 编号 | 标签页 | 做什么 |
|------|--------|--------|
| ① | **分析工作台** | 自选股列表；选一只看 K 线、财务对比、所属板块；**生成可读简报**、行动路线 |
| ② | **搜索添加** | 全球证券搜索（中文名、代码、拼音、美股代码等）→ 加入自选股 |
| ③ | **板块行情** | 行业 / 概念 / 地域 / 资金流等板块榜单，可点龙头加入自选 |
| ④ | **全球股市** | A 股 / 港股 / 美股涨跌幅、成交额等榜单；可选一只做深度溯源 |
| ⑤ | **异动全景** | 批量扫描异动，汇总原因与评分 |
| ⑥ | **行动路线** | 完整版异动解读报告（日常建议在 ① 里看并导出） |
| ⑦ | **历史记录** | 查询日志与快照恢复（数据保存在本机 `data/` 目录） |

**核心产出**：在 **① 分析工作台** 点击 **「生成可读简报」**，可预览并下载 `.md` 文件，方便复制或发给同事。

## 3. 典型使用场景

### 场景 A：早上快速过一遍自选

1. 打开应用并登录  
2. 进入 **① 分析工作台**，从自选股里选一只  
3. 看 K 线、涨跌幅、副图指标  
4. 需要分享时点 **生成可读简报** → 下载 Markdown  

### 场景 B：发现新标的

1. 在 **② 搜索添加** 输入「茅台」「SNX」「0700.HK」等关键词  
2. 点 **全球搜索**，从结果里选一只 → **加入自选股**  
3. 回到 **① 分析工作台** 继续分析  

### 场景 C：看板块热点

1. 打开 **③ 板块行情**，刷新行业 / 概念 / 资金流等榜单  
2. 对感兴趣的龙头点 **加入自选股**  
3. 在 **①** 里做进一步 K 线与财务对比  

### 场景 D：跟全球榜单异动

1. 在 **④ 全球股市** 选市场（A 股 / 港股 / 美股）和榜单类型  
2. 点 **刷新榜单**，选一只做 **深度溯源** 或加入自选  
3. 在 **⑤ 异动全景** 看批量扫描结果（可选）  

### 场景 E：发给同事（公网）

1. 管理员按 [DEPLOY_STREAMLIT.md](../DEPLOY_STREAMLIT.md) 部署到 Streamlit Cloud  
2. 把链接和密码发给同事  
3. 同事浏览器打开即可，无需安装 Python

## 4. 数据来源

本工具**不建自己的行情库**，运行时从公开接口拉取数据。主要来源如下：

| 数据类型 | 主要来源 | 说明 |
|----------|----------|------|
| A 股搜索、K 线、部分榜单 | **东方财富**（东财）公开接口 | 代码形如 `600519`、`920161`（北交所）等 |
| A 股榜单备用 | **新浪财经** | 东财不可用时兜底 |
| 港股 / 美股 / ETF | **Yahoo Finance**（通过 `yfinance` 库） | 如 `0700.HK`、`AAPL` |
| 新闻聚合 | 多源 RSS / 公开 feed | 用于异动解读中的资讯摘要 |
| 板块、F10 摘要 | 东财相关接口 | 以源站展示为准 |

**术语说明**

- **K 线**：一段时间里开盘价、收盘价、最高、最低价的图表，常用来观察走势。  
- **榜单**：按涨跌幅、成交额等排序的股票列表。  
- **简报**：把当前标的的关键信息整理成 Markdown 文档，方便阅读与转发。

数据有**延迟**（通常分钟级或更长），以各数据源实际更新为准；非交易时段可能无最新盘中数据。

## 5. 局限与免责声明

请务必了解以下限制：

1. **非投资建议**  
   所有评分、异动解读、行动路线均为**规则与公开信息推演**，仅供学习讨论，不构成买卖建议。

2. **数据准确性**  
   第三方接口可能延迟、缺字段或临时不可用；港美股依赖 Yahoo，A 股依赖东财/新浪，**不保证 100% 准确**。

3. **访问与频率**  
   公开接口可能有访问频率限制；短时间大量刷新可能失败，请稍后重试。

4. **历史与隐私**  
   - 本地运行时，自选股与部分分析快照保存在本机 `data/` 目录。  
   - 部署到 Streamlit Cloud 时，会话数据在云端实例上，**重启应用可能丢失**；敏感信息请勿写入。

5. **功能仍在演进**  
   P1 侧重「能读懂、能导出」；一键分析、页面拆分等见 [EVOLUTION.md](../EVOLUTION.md)。

6. **合规**  
   对外公开展示或商用前，须自行补齐金融信息传播、用户隐私等合规要求。

## 6. 公网访问

同事**没有 Python 环境**时，可将应用部署到 **Streamlit Community Cloud**（免费档即可内测）：

| 项目 | 说明 |
|------|------|
| 入口 | https://share.streamlit.io |
| 仓库路径 | 本仓库 `stock-assistant/app.py` |
| 访问控制 | 在 Secrets 中设置 `STOCK_ASSISTANT_PASSWORD`，用户打开链接后输入密码登录 |
| 详细步骤 | 见 [DEPLOY_STREAMLIT.md](../DEPLOY_STREAMLIT.md) |
| 操作手册 | 见 [USER_GUIDE.md](./USER_GUIDE.md) 第 5 节 |

部署成功后，你会得到形如 `https://xxx.streamlit.app` 的链接；把链接和密码发给同事即可。

**其他方式**（进阶）：Railway / Render 自建、或本机 `cloudflared` / `ngrok` 临时隧道——见项目 [README.md](../README.md) 中的「公网部署」小节。
