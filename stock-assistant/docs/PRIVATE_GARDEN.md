# 私人选股花园 · 部署指南

> **目标：** 在公网上有一个 **只有你知道密码** 的空间，免费、可持续升级，每晚来一眼「种子长了多少」。

## 这是什么

- 打开就是 **一页**：点「刷新今日 A 股推荐」，自动从公开涨幅榜 + 技术评分筛出最多 5 只。
- **推荐成绩单**：过几天点「核对」，看之前推荐在持有期内涨没涨。
- **专家模式**：侧边栏可进原版七页（搜索、K 线、导出等），平时不用管。

## 免费私密部署（Streamlit Cloud）

1. 代码已在 GitHub：`jushuolot/jinshouzhi`，子目录 `stock-assistant/`。
2. 打开 [share.streamlit.io](https://share.streamlit.io)，用 GitHub 登录。
3. New app → 选仓库 → **Main file path** 填：`stock-assistant/app.py`。
4. **Secrets** 里写（别人看不到）：

```toml
[passwords]
default = "你的强密码只给自己"
```

5. Deploy。链接只有知道密码的人能进 —— 这就是你的 **私人花园**。

## 本地私人版

```bash
cd stock-assistant
cp .streamlit/secrets.toml.example .streamlit/secrets.toml
# 编辑密码后：
./启动网页.command
```

浏览器打开 `http://localhost:8501`，同样密码门。

## 每晚「佛祖查岗」清单

1. 打开花园 → 看 **今日可关注** 表格（绿=买入，黄=观望）。
2. 点 **核对推荐成绩单**，看命中率。
3. 展开 **成长日记**，看版本号 / 进化步数是否又涨了。
4. （可选）侧边栏 **进入专家模式** 做深度分析。

## 关于「吹嘘选股能力」

- 系统会 **记录每次推荐**，并在持有期（约 2–5 天）后核对涨跌。
- **命中率**显示在页面上 —— 用真实历史说话，不是口头保证。
- 公开数据 + 规则评分 **不能** 保证每只都涨；请勿 all-in，仅供学习交流。

## 基金怎么办

- 主战场是 **A 股**；场内 ETF（如 510300、159915）可当「基金」用代码搜索加入自选。
- 场外基金暂无净值自动扫全网，后续可进化。
